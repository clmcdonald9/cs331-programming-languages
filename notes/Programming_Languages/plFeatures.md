# Identifiers and values

identifier is name of something in sc

identifiers are in some neamespace

overloading is when a single name in a single namespace refers to two or more dif entities

value is a num or string or bool or some kind of object
expression is an entity that has a value
literal is a representation of a fixed value in sc

variable is an identifier that can be associated with a val. making this association is called binding, a variable is bound to a value

identifiers have a scope
a value has a lifetime
